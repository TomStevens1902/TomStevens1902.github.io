Pacman clone (Unity C#)

Pacman is a well known arcade game that has been recreated numerous times. My intention behind creating a clone was to develop my skills in Unity as well as C# in a project where I could mainly focus on the inclusion of AI whilst developing the game for use on a mobile platform.

AI is a area where I find myself drawn to and would like to potentially pursue a career with and so throughout my degree I have developed a knowledge of skills and techniques which I have primarily used within Unreal with the use of behavior trees. In this project I intended to demonstrate the use of a state machine in C# as well as techniques used to create the various behaviors.

As well as this I previously, hadn’t attempted to develop a game for the mobile platform with my experience mainly catering to desktop applications. Because of this I needed to consider the limitations here with the screen space and mainly inputs often requiring building and testing with my phone. This is a workflow that is different to that of the standard desktop development and allowed me to get a better knowledge of the tools Unity provides.

Development in Unity is much different to Unreal and creating a project with Unity allowed me to get a better understanding of translating my knowledge of programming to another language and workflow which I feel provided me with new problems to solve and different ways of looking at structuring development.
